# 🦅 AeroBrif Gemini
Sistema inteligente de briefing aeronáutico que utiliza a **IA Gemini 1.5 Flash** para analisar condições de voo, calcular horários solares e planejar rotas VFR/IFR.

### ✨ Funcionalidades
- 🗺️ Mapa dinâmico com Leaflet.
- 🤖 Briefing gerado por Inteligência Artificial.
- ☀️ Cálculo de Nascer e Pôr do Sol automático.
- ⌚ Planejamento de ETA (Estimated Time of Arrival).
